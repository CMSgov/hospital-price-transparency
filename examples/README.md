# Overview of the Example MRFs: Last Updated on 04/29/2026

CMS has provided three example MRFs for the fictional West Mercy Hospital that contracts with two fictional payers/plans, Region Health Insurance HMO and Platform Health Insurance PPO. There is one example file in each of the allowed formats, Wide CSV, Tall CSV, and JSON, and they all depict the same fictional data. These files are meant to assist hospitals with implementation of the hospital price transparency requirements. The files can be found in the CSV and JSON subfolders of this folder, and they all pass the [Validator](https://cmsgov.github.io/hpt-tool/online-validator/) against the current requirements. The Validator helps hospitals ensure adherence to CMS form and manner requirements. The CSV examples present the data headers in row three in an order that differs from the CSV templates. Changing the order of the data headers will not generate a deficiency.

The following table lists scenarios hospitals may face in the "Scenario" column. The following three columns list where a hospital can find the scenario depicted in the Wide CSV example file, the Tall CSV example file, and the JSON example file, respectively. CMS welcomes suggestions for broadly applicable scenarios to be added to the files. Suggestions can be emailed to <PriceTransparencyHospitalCharges@cms.hhs.gov>.
